# Python Practice Projects

Collection of console-based Python practice scripts for logic, algorithms, and OOP basics.

## Folder Structure

- `Calculator/`
- `Greedy Algorithm/`
- `Number Sync/`
- `Versity Project/`

## 1) Calculator

Path: `Calculator/calculator.py`

Features:
- Evaluates user-provided math expressions via `eval`.
- Stores previous calculations in memory.
- Supports recall of recent calculation history.

Run:

```bash
python "Calculator/calculator.py"
```

## 2) Greedy Algorithm

Path: `Greedy Algorithm/`

Scripts:
- `greedy algorithm v1.py`: minimal coin-count for fixed denominations.
- `greedy algorithm v1(random input).py`: user-provided denominations + minimal count.
- `greedy algorithm v2.py`: fixed denominations with per-denomination count output.
- `greedy algorithm v2(random input).py`: user-provided denominations with detailed count output.

Run example:

```bash
python "Greedy Algorithm/greedy algorithm v2.py"
```

## 3) Number Sync

Path: `Number Sync/number sync.py`

Features:
- Prints numbered lines with zero-padding based on max value length.
- Format example: `001.Name`

Run:

```bash
python "Number Sync/number sync.py"
```

## 4) Versity Project

Path: `Versity Project/`

Scripts:
- `bank.py`: CLI bank simulation with account creation, PIN authentication, deposit, withdraw, and history.
- `calculator.py`: menu-based basic calculator with arithmetic, average, and square root.

Run examples:

```bash
python "Versity Project/bank.py"
python "Versity Project/calculator.py"
```

## Requirements

- Python 3.x
- Standard library only for these scripts

## Notes

- These scripts are learning-focused and mostly interactive.
- Some scripts intentionally keep logic straightforward rather than production-hardened.
